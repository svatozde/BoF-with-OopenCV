# VMM Bag of Features

School project

### Installing

Server use flask framwork
Installing step by step

```
python -m pip install Flask
```

## Running 
```
$ export FLASK_APP=BoF-server.py
```
```
$ flask run
```
You can use debug mode for logs
```
$ export FLASK_DEBUG=1
```
